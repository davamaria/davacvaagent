# Source Priority

Search in this order unless the user names a specific source:

1. Workspace folders such as `knowledge-base/`, `case-studies/`, `notes/`, or `exports/`
2. Structured markdown, text, PDF, or spreadsheet exports supplied by the user
3. Connected enterprise tools if they are available in the current Codex session
4. General cloud guidance only as a fallback after stating that Endava-specific case-study evidence was not found

## Query construction

Combine the domain plus delivery pattern:

- `platform engineering developer portal golden path`
- `landing zone azure enterprise migration`
- `aws modernization regulated workload`
- `gcp migration data platform`
- `finops cost optimization enterprise`

Broaden incrementally if the first pass is too narrow.
