# Deliverables

Store all agent-generated client outputs in this directory.

## Naming convention

Prefix every output file with the client name.

## Examples

- `enshite-plc-migration-roadmap.md`
- `enshite-plc-wave-plan.xlsx`
- `enshite-plc-landing-zone-target-architecture.md`
- `enshite-plc-executive-summary.pptx`

## Rule

When generating deliverables for a client, save them in `deliverables/` and start the filename with the client name in lowercase kebab-case.
